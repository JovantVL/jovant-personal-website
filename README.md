# Jovant's Personal Website

## Overview
This repository contains my personal website project created for the User Interface Design course assignment.
The website presents my personal profile, interests, organizational journey, and contact information while applying fundamental UI/UX principles such as consistency, visual hierarchy, responsive design, and usability.

## UI/UX Principles Applied
### Consistency
Consistent color palette, typography, spacing, and card design throughout the website.
### Visual Hierarchy
Important information such as name, role, and contact information are emphasized using size, contrast, and spacing.
### Accessibility
Readable typography, sufficient spacing, and responsive layout.
### Responsive Design
Optimized for desktop, tablet, and mobile devices.
